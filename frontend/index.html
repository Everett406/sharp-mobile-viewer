技术说明

SharpView 利用 Apple 的 SHARP 模型，通过 GitHub Actions 云端推理，将单张照片重建为 3D 高斯泼溅场景。推理结果以 .ply 格式存储，支持本地实时渲染查看。

模型大小: 2.44 GB 推理时间: 5-15 分钟 输出格式: .ply
